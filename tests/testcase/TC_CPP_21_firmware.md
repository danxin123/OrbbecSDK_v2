## 关联功能点

本模块覆盖 [sdk_feature_matrix.md](../sdk_feature_matrix.md) 中 **模块 21：固件管理** 的全部功能点：

| 编号 | 功能点 | C API | C++ API |
|------|--------|-------|---------|
| 21-010 | 从文件升级固件 | `ob_device_update_firmware(dev, path, cb, async)` | `updateFirmware()` |
| 21-020 | 从缓冲升级固件 | `ob_device_update_firmware_from_data()` | `updateFirmwareFromData()` |
| 21-030 | 更新深度预设 | `ob_device_update_optional_depth_presets()` | `updateOptionalDepthPresets()` |
| 21-040 | 重启设备 | `ob_device_reboot()` | `reboot()` |
| 21-050 | 全局时间戳支持 | `ob_device_is_global_timestamp_supported()` | `isGlobalTimestampSupported()` |
| 21-060 | 启用全局时间戳 | `ob_device_enable_global_timestamp()` | `enableGlobalTimestamp()` |
| 21-070 | 设备状态查询 | `ob_device_get_device_state()` | `getDeviceState()` |
| 21-080 | 状态变化回调 | `ob_device_set_state_changed_callback()` | `setStateChangedCallback()` |
| 21-090 | 心跳监测 | `ob_device_enable_heartbeat()` | `enableHeartbeat()` |
| 21-100 | 原始厂商命令 | `ob_device_send_and_receive_data()` | `sendAndReceiveData()` |
| 21-110 | 相机标定参数 | `ob_device_get_calibration_camera_param_list()` | `getCalibrationCameraParamList()` |
