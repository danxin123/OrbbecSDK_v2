## 关联功能点

本模块覆盖 [sdk_feature_matrix.md](../sdk_feature_matrix.md) 中 **模块 23：日志系统** 的全部功能点：

| 编号 | 功能点 | C API | C++ API |
|------|--------|-------|---------|
| 23-010 | 设置日志级别 | `ob_set_logger_severity(severity)` | 对应方法 |
| 23-020 | 输出到文件 | `ob_set_logger_to_file(severity, dir)` | 对应方法 |
| 23-030 | 设置文件名 | `ob_set_logger_file_name(filename)` | 对应方法 |
| 23-040 | 输出到控制台 | `ob_set_logger_to_console(severity)` | 对应方法 |
| 23-050 | 输出到回调 | `ob_set_logger_to_callback(severity, cb)` | 对应方法 |
| 23-060 | 外部消息注入 | `ob_log_external_message(...)` | 对应方法 |
