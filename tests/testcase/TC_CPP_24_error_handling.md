## 关联功能点

本模块覆盖 [sdk_feature_matrix.md](../sdk_feature_matrix.md) 中 **模块 24：错误处理** 的全部功能点：

| 编号 | 功能点 | C API | C++ API |
|------|--------|-------|---------|
| 24-010 | 获取错误状态码 | `ob_error_get_status()` | 抛出 `ob::Error` 异常 |
| 24-020 | 获取错误消息 | `ob_error_get_message()` | `what()` |
| 24-030 | 获取出错函数 | `ob_error_get_function()` | `getFunction()` |
| 24-040 | 获取异常类型 | `ob_error_get_exception_type()` | `getExceptionType()` |

### 基本信息

| 字段 | 值 |
|------|-----|
| 编号 | TC_CPP_24_01 |
| 名称 | 异常类型与信息 |
| 目的 | 验证异常对象包含完整信息 |
| 覆盖功能点 | 24-010~24-040 |
| 前置条件 | SDK 可用 |
| 硬件依赖 | 否 |
| 优先级 | P0 |

### 测试步骤

1. 触发一个异常（如无效参数）
2. 捕获 `ob::Error` 异常
3. 读取异常信息

### 检查项

- ✅ `getStatus()` 非零
- ✅ `what()` 非空
- ✅ `getFunction()` 非空（出错的 API 函数名）
- ✅ `getExceptionType()` 返回有效枚举

### 预期结果

异常对象包含完整错误信息。
