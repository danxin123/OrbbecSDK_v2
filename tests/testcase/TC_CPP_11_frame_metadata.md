## TC_CPP_11_01 — 元数据存在性检查与基本读取

### 用例版本记录

| 版本 | 日期 | 变更说明 |
|------|------|---------|
| 1.0 | 2026-03-17 | 初始版本 |

### 基本信息

| 字段 | 值 |
|------|-----|
| 编号 | TC_CPP_11_01 |
| 名称 | 元数据存在性检查与基本读取 |
| 目的 | 验证元数据存在性检查和基本读取功能 |
| 覆盖功能点 | 11-010, 11-020 |
| 前置条件 | 至少一台设备通过 USB 连接 |
| 硬件依赖 | 是 |
| 优先级 | P0 |

### 测试步骤

1. 启动 Pipeline 获取帧
2. 调用 `frame->hasMetadata(OB_FRAME_METADATA_TYPE_TIMESTAMP)`
3. 若存在，调用 `frame->getMetadataValue(OB_FRAME_METADATA_TYPE_TIMESTAMP)`

### 检查项

- ✅ `hasMetadata(TIMESTAMP)` 返回 true（若设备支持）
- ✅ `getMetadataValue(TIMESTAMP)` 返回值 > 0
- ✅ 元数据值与 `frame->timeStampUs()` 一致或相关

### 预期结果

元数据存在性检查准确，读取的值有效。
